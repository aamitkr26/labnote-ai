# LabNote-AI

## What this is
A mobile-first lab notebook + AI assistant for students and researchers.

Core focus:
Write → Think → Export

This is NOT:
- LIMS
- IoT platform
- Hardware-integrated system
- Collaboration tool

## MVP Scope (Locked)
- Structured lab notes (Markdown-based)
- Local-first storage (no accounts)
- Context-aware AI assist (single provider)
- Export: PDF / Markdown / Text

Anything outside this is out of scope.

## Architecture
Refer to /docs/architecture.md

## Decisions Log
- No authentication
- No sync
- No images
- No multi-provider AI
- No backend database
